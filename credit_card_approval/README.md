# 💳 Credit Card Approval Prediction using Machine Learning

## 📌 Project Overview

This project is a Machine Learning-based web application that predicts whether a credit card application will be **Approved** or **Rejected** based on the applicant's information. The application is developed using **Python**, **Flask**, and a trained Machine Learning model to provide quick and accurate predictions through a simple web interface.

---

## 🚀 Features

- Predicts credit card approval status
- User-friendly web interface
- Machine Learning-based prediction
- Fast and accurate results
- Simple and responsive design

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/Credit-Card-Approval-Prediction.git
```

### Go to the Project Folder

```bash
cd Credit-Card-Approval-Prediction
```

### Install Required Packages

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

### Open Your Browser

```
http://127.0.0.1:5000
```
